# BE Final Year Project

## Team Number
Team_59

## Project Title
AR-Integrated Medical Learning Platform

## Project Description
This project focuses on developing an AR-based learning platform to help medical students visualize and understand human anatomy and medical concepts using 3D models and interactive technologies.

## Technologies Used
- Three.js / WebAR
- AI Models (for assistance and learning support)

## Repository Structure
- `/src` – Source code
- `/public` – Assets and models
- `/docs` – Documentation

## Guide
This project is uploaded as part of the BE Final Year Project submission to the college GitHub repository.
